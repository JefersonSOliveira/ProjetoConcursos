Questão 001

Gabarito: Questão Correta

MIB (Managemente information base - base de informações de gerenciamento) é um banco de dados usado para gerenciamento de entidades em uma rede de comunicação. Ele é frequentemente utilizado para se referir a um subconjunto particular, mais corretamente referenciado como módulo MIB. Frequentemente se associa ao SNMP (Simple Network Management Protocol).

→ MIB privada fornece informações específicas como configurações e colisões, mas também reinicializa/desabilita uma ou mais portas de um roteador.

• A principal informação que este tipo de gerenciamento realiza é informar se o dispositivo gerenciado está UP (funcionando) ou DOWN (falhando)
