Questão 009

Gabarito: Questão Correta

CISC (sigla para Complex Instruction Set Computer, ou, em uma tradução literal, "Computador com um Conjunto Complexo de Instruções") é um processador capaz de executar centenas de instruções complexas diferentes sendo, assim, extremamente versátil.

RISC (Reduced Instruction Set Computer ou Computador com um Conjunto Reduzido de Instruções) é uma linha de arquitetura de computadores que favorece um conjunto simples e pequeno de instruções que levam aproximadamente a mesma quantidade de tempo para serem executadas.
